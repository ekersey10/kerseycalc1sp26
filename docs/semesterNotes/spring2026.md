# Notes about Spring 2026

The materials mostly worked well this semester.  A few suggested changes for next time:

- We need to switch learning targets 14 and 16, as they are out of order and this is confusing.
- The grade posted on Canvas should include the learning target quizzes, which it currently does not. It only gives the participation half. 
- I would consider making LT quizzes worth fewer points for subsequent attempts so that students take them more seriously. If they are actual scores incorporated into Canvas, I would just have it worth one fewer point (out of 10) with each subsequent attempt. The current rubric for the LT quizzes does not affect their grade, but the score for Partial Understanding is a 5. It might be good to make this the minimum possible score, so students would only take any given quiz up to four times. Just a thought. 
- I brought copies of the LT quizzes from the last two weeks to class every Tuesday. They could retake older ones, but only if they specifically requested it ahead of time or if they came to student hours. There were also maybe three Tuesdays throughout the semester when I brought copies of all previous LT quizzes.
